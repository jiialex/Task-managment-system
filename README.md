Task Management System

A full-stack **Task Management System** built with **NestJS** (backend) and **React** (frontend).  
This project allows users to create, assign, and track tasks with deadlines, notifications, and role-based access
 Features
 Authentication: Login & Signup with JWT
User Management: Create and manage users with roles (Admin, Member)
Task CRUD: Create, update, delete, and mark tasks as complete
Deadlines: Assign due dates and track progress
Notifications: Email or in-app reminders for upcoming deadlines
Dashboard: Overview of tasks by status (Pending, In Progress, Completed)
Profile: Upload and manage profile photos

 Tech Stack
| Layer        | Technology |
|--------------|------------|
| Frontend     | React, Vite, css|
| Backend      | NestJS, TypeScript |
| Database     | PostgreSQL (via Prisma/TypeORM) |
| Auth         | JWT, bcrypt |
| Deployment   | Vercel/Render |
